🟢 **Phase 1 — 프로젝트 초기화** (필수)

Next.js 16 + TypeScript 기반 포트폴리오 프로젝트를 초기화합니다.
초기화 단계에서는 UI 스타일이 고정되지 않으며, 이후 `/theme-on` 커맨드에서 스타일 테마를 선택·설치할 수 있습니다.
이 command는 **가장 먼저 실행되어야 합니다**.

## 목표

- Next.js 16 + React 19 프로젝트 구조 생성
- 필수 의존성 설치 (styled-components, Framer Motion 등)
- 멀티 테마 라우팅 폴더 구조 자동 생성 (2X.X.md 파일을 모두 포함하도록 함)
- 환경 변수 템플릿 생성
- TypeScript, ESLint, Prettier 설정

## Steps

#### 3-1. 테마 라우트 디렉토리 생성

`.claude/commands/` 디렉토리에서 `2a.`, `2b.`, `2c.`… 패턴의 테마 명령 파일을 스캔하여 테마 목록을 동적으로 추출합니다.

**테마 목록 추출 방법:**
1. `.claude/commands/2[a-z]*.md` 파일들을 Glob으로 검색
2. 각 파일 내 `src/app/{테마}/layout.tsx` 패턴에서 라우트 이름 추출
3. 추출된 테마 이름으로 디렉토리 생성

```bash
mkdir -p src/styles/themes
# 추출된 각 테마에 대해:
mkdir -p src/app/{테마이름}
```

각 테마 라우트에 빈 `layout.tsx` 플레이스홀더를 생성합니다.
실제 내용은 `/theme-on <테마이름>` 실행 시 채워집니다.

```typescript
// src/app/{테마이름}/layout.tsx  (플레이스홀더 — /theme-on 실행 후 교체됨)
export default function ThemeLayout({ children }: { children: React.ReactNode }) {
  return <>{children}</>;
}
```

#### 3-2. 핵심 파일 생성

**`src/proxy.ts`** 생성 (최소 버전):

> ⚠️ Next.js 16에서는 `middleware.ts` 대신 `proxy.ts`를 사용합니다. `middleware.ts`는 deprecated 경고가 발생합니다.

```typescript
import { NextRequest, NextResponse } from "next/server";

export function proxy(request: NextRequest) {
  if (request.nextUrl.pathname.startsWith("/admin")) {
    return NextResponse.next();
  }

  if (request.nextUrl.pathname.startsWith("/_next")) {
    return NextResponse.next();
  }

  return NextResponse.next();
}

export const config = {
  matcher: ["/((?!api|_next/static|_next/image|favicon.ico).*)"],
};
```

**`src/lib/memory-db.ts`** 생성:

```typescript
interface PageLog {
  id: number;
  url: string;
  path: string;
  referrer?: string;
  userAgent?: string;
  ipAddress?: string;
  timestamp: Date;
}

let logs: PageLog[] = [];
let nextId = 1;

export const memoryDB = {
  addLog: (data: Omit<PageLog, "id" | "timestamp">) => {
    const log: PageLog = {
      id: nextId++,
      ...data,
      timestamp: new Date(),
    };
    logs.push(log);
    return log;
  },
  getLogs: (limit: number = 100) => {
    return logs
      .sort((a, b) => b.timestamp.getTime() - a.timestamp.getTime())
      .slice(0, limit);
  },
  getStats: () => {
    const stats: Record<string, { count: number; lastVisit: Date }> = {};
    logs.forEach((log) => {
      if (!stats[log.path])
        stats[log.path] = { count: 0, lastVisit: log.timestamp };
      stats[log.path].count++;
      if (log.timestamp > stats[log.path].lastVisit) {
        stats[log.path].lastVisit = log.timestamp;
      }
    });
    return stats;
  },
};
```

**`src/actions/logging.ts`** 생성:

```typescript
"use server";

import { cookies } from "next/headers";
import { memoryDB } from "@/lib/memory-db";

export async function logPageView() {
  try {
    const cookieStore = await cookies();
    const pageViewData = cookieStore.get("_pageViewData")?.value;
    if (!pageViewData) return;

    const data = JSON.parse(pageViewData);
    memoryDB.addLog({
      url: data.url,
      path: data.path,
      referrer: data.referrer,
      userAgent: data.userAgent,
      ipAddress: data.ipAddress,
    });
  } catch (error) {
    console.error("Failed to log page view:", error);
  }
}
```
