# 💎 Full-Side Engineer Portfolio: Jihye "Toss-Style" Lee

Next.js 16 + Tailwind CSS v4 기반의 **토스(Toss) 인터페이스** 포트폴리오 시스템

> **"기술의 혁신이 서비스의 성장으로 이어지도록."**
> 20년의 엔지니어링 내공을 바탕으로 마타수학의 AI 서비스 리뉴얼을 주도하고, 압도적인 개발 효율성을 증명했습니다.

## 🎨 Design Concept: "Extreme Simplicity"

- **Typography**: Toss Product Sans / Pretendard (가독성 최우선)
- **Visual**: 여백의 미, 그림자(Shadow)를 활용한 입체감, 토스 특유의 부드러운 애니메이션
- **UX Flow**: 나열형 경력이 아닌 **'마타수학 리뉴얼'**을 필두로 한 핵심 성과 타임라인

## 🛠 Tech Stack (High Performance)

- **Framework**: Next.js 16 (App Router), React 19

- **Styling**: Tailwind CSS v4, Framer Motion, Styled-Components

- **State/Data**: React-Query, Recoil, Redux (배민상회 어드민 데이터 패칭 최적화 로직 적용)

- **Efficiency**: AI 자동화 워크플로우 (Claude Code, Gemini, GCP Vertex AI 기반)

## 📂 Project Structure (Toss Service Architecture)

```
src/
├── app/
│   ├── (home)/            # 토스 홈 UI: 마타수학 리뉴얼 등 주요 성과 지표 요약
│   ├── projects/          # [보완] 마타수학 집중 탐구: 전략 제안부터 프론트엔드 구현까지의 Full-Side 기록
│   ├── efficiency/        # '15명 분의 일을 3명이 해결한' 리소스 최적화 및 AI 자동화 기술 상세
│   └── resume/            # 20년 경력의 정수만 담은 텍스트 기반 이력서
├── components/            # 토스 디자인 시스템(TDS) 스타일의 컴포넌트
├── actions/               # Server Actions
└── data/                  # resume.json (마타수학 프로젝트 핵심 데이터 중심)

```

## 📈 Impact-Driven Content (Focus: Mata-Edu)

억지스러운 마케팅 용어 대신, **기술적 성취가 가져온 실질적 서비스 변화**를 기술합니다.

- **[Case 1] AI 마타수학 서비스 리뉴얼 주도 (2026)**
- _Impact_: 서비스 전략 제안 및 리뉴얼 구조 설계를 주도하여 AI 학습 서비스의 사용자 경험을 근본적으로 개선했습니다.

- _Technical_: React/Next.js 기반의 안정적인 UI/UX 구현 및 기획-개발 브릿지 역할 수행.

- **[Case 2] 10배의 리소스 효율 증명**
- _Impact_: 전통적인 조직에서 15명이 6개월간 수행할 규모를 3명의 정예 인원으로 3개월 만에 완수했습니다.
- _Technical_: AI 디자인 시스템 자동화 및 Python 기반 생성 자동화 로직으로 개발 생산성을 극대화했습니다.

- **[Case 3] 기술 기반 서비스 확장**
- _Impact_: 초등 AI 마타수학(2025) 및 수학 모둠활동 '패들리'의 전략을 제안하고 MVP를 빠르게 시장에 안착시켰습니다.

- _Technical_: Figma 인터랙션 설계부터 Vercel 배포까지 전 과정을 주도했습니다.

## 🤖 AI Execution Commands

- `/toss-theme-on`: 토스 특유의 화이트톤 UI와 블루 포인트 컬러를 적용합니다.
- `/mata-focus-mode`: 마타수학 프로젝트를 최상단에 배치하고 상세 기술 스택을 강조합니다.
- `/impact-summary`: 20년의 경력을 '기술 기반의 실행 역량' 중심으로 요약합니다.
- `/resource`: 폴더내에 있는 서비스 대표화면을 적용합니다.
